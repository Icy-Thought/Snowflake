* Installation
** Doom-Emacs
#+begin_src sh
git clone --depth 1 https://github.com/hlissner/doom-emacs ~/.emacs.d
~/.emacs.d/bin/doom install
#+end_src

** BMC (Math)
#+begin_src sh
if [[ ! -d "/path/to/dir" ]]; then
   mkdir -p ~/texmf
fi
git clone --depth 1 https://github.com/tecosaur/BMC.git ~/texmf
texhash
#+end_src
